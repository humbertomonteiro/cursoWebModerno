Tag table

Para criar uma tabela usamos a tag table, alguns programadores usar a tag caprion para referenciar a que se refere a tabela criada, mas eu particularmente não uso. Uma tabela tem o cabeçalho, o corpo e o rodapé, são essas as maiores partes da sua tabela. Então dentro da tabela usaremos a tag thead para informar onde será o cabeçalho, dentro dessa tag usaremos a tag tr que será uma linha e dentro dessa linha usaremos a tag th para informarmos o cada parte do cabeçalho da tabela.

Agora vamos usar depois da tag thead a tag do corpo da tabela, a tag tbody, tabem vamos usar uma linha para colocar os dados da tabela com a tag td. No exemplo que será mostrado terá duas linhas com dois produtos, que mostrará o que foi colocado na tr do cabeçalho. E por último vamos fazer o rodapé com a tag tfoot que terá apenas uma linha (tag tr) e nessa linha só mostrará o nome total e o valor total.

Perceba que a palavra total do rodapé esta usando 3 espaços. Para fazer isso é simples, vamos apenas colocar um parametro na td que queremos que use mais espaços. O parametro é colspan com a quantidade de colonas que quer ocupar com o espaço dessa td.
